Overview

The AddCredential component provides a comprehensive interface for adding new credentials to the RDMasterX system with advanced security features and password generation capabilities.

‚ú® Features

  • Required Fields:
    • Username - The account username
    • Password - Can be manually entered or generated
    • Description - Brief description (e.g., "Zabbix monitoring account")
  • Optional Fields:
    • URL - Website address (e.g., "https://zabbix.example.com")
  • Password Generator:
    • Customizable length (4-50 characters)
    • Include/exclude uppercase letters
    • Include/exclude lowercase letters
    • Include/exclude numbers
    • Include/exclude special characters
    • One-click generation and copy to clipboard
  • Security Features:
    • Password strength indicator
    • Password visibility toggle
    • AES-256 encryption
    • Copy to clipboard functionality
  • User Experience:
    • Dark/Light theme support
    • Real-time validation
    • Responsive design
    • Intuitive interface
